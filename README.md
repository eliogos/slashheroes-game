# SLASHHEROES

A text-based dungeon crawler Discord bot built with Cloudflare Workers. Explore dungeons, fight enemies, collect loot, and form parties with friends using Discord's slash commands and User Apps support.
This README.md is currently generated by AI as a summary of the whole project. It is a placeholder or base template. 

## Features

- 🗡️ **Dungeon Exploration** - Explore procedurally generated dungeons with unique rooms and encounters
- 👥 **Party System** - Team up with friends in Group DMs
- 🎮 **User Apps Support** - Works in both servers and private channels
- 🏰 **Dynamic Room Generation** - Unique dungeon layouts based on server and channel IDs
- 🛡️ **Hero System** - Create and customize your hero with an onboarding flow
- ⚔️ **Combat & Loot** - Fight monsters and collect treasures

## Commands

- `/explore` - Explore the current dungeon location
- `/party` - Send an invite to your party (Group DM only)
- `/about` - Learn more about SLASHHEROES
- `/ping` - Check bot responsiveness

## Tech Stack

- **Runtime**: Cloudflare Workers
- **Database**: Cloudflare D1 (SQLite)
- **Language**: JavaScript (ES Modules)
- **Discord API**: discord-api-types v0.38.34
- **Testing**: Vitest with Cloudflare Workers pool
- **Deployment**: Wrangler

## How It Works

### Dungeon Generation

SLASHHEROES uses a unique procedural generation system that creates consistent dungeon layouts based on:
- Server ID
- Category ID (or server ID if no category)
- Channel ID

This ensures players in the same channel always see the same room, creating a shared exploration experience.

### Anti-Abuse Protection

The bot includes several protection mechanisms:
- Snowflake age verification (3-day minimum for servers/channels)
- Dead-end detection to prevent infinite exploration
- Signature verification for all Discord interactions

### User Apps Support

The bot works in multiple contexts:
- Guild (Server) channels
- Private channels and DMs
- Group DMs for party play

## Database

The bot uses Cloudflare D1 for persistent storage. Key features:
- Hero data storage
- Party management
- Exploration history
- Loot inventory
