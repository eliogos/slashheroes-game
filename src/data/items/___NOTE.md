# Item Data Notes

The use of measurements, units, and formula-style values in this data layer is mainly for **semantics** and for incorporating **real-world scientific and mathematical concepts** to support a sense of **semi-realism**.

These values are not intended to serve as a strict simulation of reality. Instead, they help make the item system more readable, internally consistent, and easier to balance through believable logic.

## In practice
- Units such as weight, length, speed, and volume provide clearer contextual meaning to item values
- Some formulas borrow from real-world physics or math concepts
- Gameplay balance still takes priority over strict realism

The goal is **grounded game feel**, not perfect real-world accuracy.
